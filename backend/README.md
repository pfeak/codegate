# CodeGate Backend

激活码/邀请码/验证码分发平台后端服务

## 快速开始

### 安装依赖

```bash
uv sync
```

### 运行服务

```bash
uv run python main.py
```

服务将在 `http://localhost:8000` 启动。

### 访问

- **Web 界面**: http://localhost:8000
- **API 文档**: http://localhost:8000/docs
- **健康检查**: http://localhost:8000/health

## 技术栈

- Python 3.12+
- FastAPI
- SQLAlchemy
- SQLite
