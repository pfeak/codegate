# CodeGate SDK

CodeGate SDK 提供多种编程语言的客户端库，用于与 CodeGate API 进行交互。

## 支持的语言

- **Python** - [`python/`](./python/) - 已完成
- **JavaScript/TypeScript** - `javascript/` - 计划中

## 目录结构

```shell
sdk/
├── python/          # Python SDK
│   ├── src/
│   ├── tests/
│   ├── pyproject.toml
│   └── README.md
└── javascript/      # JavaScript/TypeScript SDK（计划中）
```
